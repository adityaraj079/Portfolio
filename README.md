#### Steps for Running locally

1. Go to 
- ` api>index.py `
- Uncomment the last 2 lines

2. Create a virtual environment
- ` python -m venv venv `

3. Activate vitual environment
- ` venv\Scripts\activate `

4. Install dependencies 
- ` pip install -r requirements.txt `

5. Run the flask app
- ` python api/index.py `
